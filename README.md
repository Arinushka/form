# Тестовое задание: 3-шаговая форма (SPA)

Одностраничное приложение с тремя последовательными формами, валидацией и финальной модалкой подтверждения.

## Стек

- React + TypeScript + Vite
- React Router (`/step-1`, `/step-2`, `/step-3`)
- Bootstrap + React-Bootstrap (быстрая стилизация + готовая доступная модалка)
- React Hook Form + Zod (удобная, типобезопасная валидация)
- React IMask (маска телефона `0XXX XXX XXX`)
- TanStack React Query (кэширование списка категорий с API и переиспользование результата)

Обоснования выбора библиотек также продублированы комментарием в `src/app/providers.tsx`.

## Запуск

```bash
npm install
npm run dev
```

Открыть в браузере: `http://localhost:5173/`

## Проверка по ТЗ (кратко)

- **Шаг 1**: телефон (маска `0XXX XXX XXX`), имя, фамилия, пол — все обязательные.
- **Шаг 2**: место работы (категории с `GET https://dummyjson.com/products/categories`, кэшируется), адрес — обязательные.
- **Шаг 3**: сумма ($200…$1000, шаг 100) и срок (10…30 дней, шаг 1) — обязательные.
- **Submit**: `POST https://dummyjson.com/products/add` с `{"title": "<Имя> <Фамилия>"}` и показ модалки:

```
Поздравляем, <Фамилия> <Имя>. Вам одобрена <сумма> на <срок> дней.
```
